# Provenance

This project was initially inspired by [tianhetonghua/Charles-mcp-server](https://github.com/tianhetonghua/Charles-mcp-server).

The current `charles-mcp` codebase has since been substantially rewritten and reorganized for a different architecture and use case, and is maintained independently.

This note is intended to provide a concise factual provenance statement for the public repository history and current documentation.
